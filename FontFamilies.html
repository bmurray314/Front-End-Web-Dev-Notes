This is the serif font-family. Serif fonts have extra details on the end of the main strokes of the letters. Browers are supposed to support at least one font style of each category. In this case, it is typical to write font-family: Georgia, Times, serif. This would mean that the website should look for the Georgia font, then if that one is not installed, go the Times font, then if none of the others are installed, go to whatever serif font is installed. If a font name is multiple words, it needs to be put in double quotes. (It's considered standard practice to only choose your top two choices then to pick a generic one for the last choice.
