First Letter/Line

This is an example of a paragraph where the first line will be bold and the first letter of the paragraph will have a font-size which is 200% (double) what it normall is. These are both technically called 'pseudo-elements'. CSS thinks of these as extra elements in the code. With the element being separate, it can be styled separately to stand out. There are also peusdo-classes too!
