Font Families

The CSS code for font families is font-family. Options are different names of font families. Options such as "Courier New" (which have 2 words), are required to be put into double quotation marks. Also, the list in which the fonts are labeled is the list in which the browser will try to render them.

For example, if the CSS code was: font-family: Georgia, Times, serif, the browswer would see if Georgia was available first. If it wasn't it would move on to the next, and so on.

Font Size

Click here for an example of what different font-sizes looks like on a web page.

The CSS code for font size is font-size. Options include pixels (px), percentage based on the normal font size (%) or EMS(em). An 'em' is the equivalent to the width of the letter m.

Units of Type Size

There are 3 main units when it comes to the size of the font. Below are the 3 types and a little about each of them.

  • Pixels (px): This is exactly what you think it is, just the size of the text from the pixels on the screen. Note that with lower resolution monitors, the text may seem bigger than intended. With super-high resolution monitors, the opposite may be true.
  • Percentages (%): Typing a percentage as a font size will increase or decrease the standard font size of the web pages. In most cases, the font size is 16px. In this case, choosing a font size of 200% would double the font size to 32.
  • EMS (em): This unit is based off of the font size of the letter m. 1em = the width of the letter m. In other words, the font scales up or down depending on how big the letter m is.
